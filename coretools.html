What are the Core Tools

By Yvette Avalos

Core Tools

Programming languages change all the time, but there are three tools that every developer should master. The first tool is your Editor, it is the one application that you will have the most contact with, they become in two categories, IDE’s or text editors. IDE’s or Integrated Development Environments give you a ton of functionality, but every IDE gives you a unique and highly opinionated workflow, and often the sheer amount of options can be overwhelming, so they take longer to master. Popular IDE’s include Visiual Studio, NetBeans, Eclipse, and Android Studio. Popular editors include Sublime Text, VIM, as well as Atom. The next tool you will want to master is the Terminal. A lot of modern applications offer CLIs or Command Line Interfaces, they help you take care of common tasks for different tools. The last essential tool that you will need to master is your good old browser, browsers are where you test what you have created on your editors but they also have a lot of powerful features for developers that allow you to debug your HTML, CSS, and JavaScript and do other advanced tasks like test your animations or sync to phones for mobile testing.

More Core Tools

Besides the big three tools, there are lots of other tools that you would need to know to be the most marketable. Although you can manage database through the terminal, there are applications that let you visualize the data through a much better interface. One of the most common is PHP MyAdmin. Another tool that is popular with developers is called Version Control. It is like having a time machine for your code and lets you move through changes like an Undo that works even after you have shut dowm ur machine.The most popular system is called Git, it lets you store key changes to your projects called command. You can also compare changes between commits and create branches to test new features. In order to work with others when using Git most projects use a website called Github and it adds excellent collab features to your team. Build tools let you create workflows for common tasks, they manage things like transpiling, minifying and debugging. You should also learn some frameworks, libraries of pre-built code that help you build sites faster, and these are so common that most websites use at least one of these. Bootstrap, a framework which helps you build websites with an easy-to-learn but powerful grid system and excellent interactive features. JQuery standarizes the way that JavaScript works in different browsers and give you additional features that make it easier to build interactions.JavaScript framework like Angular.js and React provide libraries that help you build single page applications and interfaces.
