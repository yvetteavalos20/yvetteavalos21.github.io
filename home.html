Getting to know the Page

As a web designer it is important to understand how the web works and the processes involved in hosting, serving, and rendering websites.The internet and the web are not the same thing.The internet in term is used to refer to a massive network of millions of computers all over the world that used to share and transmit information.It is done through various protocols and languages and includes things like email, sms messaging, application data as well as the web itself.

Web - World Wide Web is the way of transmitting data over the Internet using the HTTP protocol and HTML.The Web can be summarized as a series of interactions between two types of Systems, Clients, and Servers.Clients are the device that request and render web content.Popular clients include browser, mobile apps, screen readers, and various content aggregators.Try to keep up with all new clients and how they access and render your content is one of the most challenging aspects of web design.

Servers are applications that deliver web content or services to clients.There is a wide range of web servers and types that specialize in specific types of data or specific processing capabilities.They are all involved in some way in delivering the content that you request or the services that you need performed.You are gonna need to become familiar with the different types of servers and how they work.
