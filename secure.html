The World of Security

If you are thinking about building an ecommerce site, you have most likely heard and thought about the importance of security. Security is important for three big reasons; You can be held financially liable for stolen credit cards. If there is a breach, you could be fined, you will lose customersâ€™ trust if there is a breach, and having a secure-looking site, will actually increase your conversions. Anytime someone is entering sensitive information, like credit cards or passwords, you should make sure that the page is secure. HTTPS encrypts website data using the Secure Socket Layers protocol, more commonly know as SSL. You need an SSL Certificate for your website to use SSL. Installing an SSL Certificate goes far beyond this course. In the interest of getting your site up and running quickly it is suggested to have your host install the SSL Certificate.

Once you get your SSL installed, you still need to force users to use SSL and prevent them from using regular HTTP when they are on a page where they share sensitive data. A much easier way to let your ecommerce platform handle it for you. There are also WordPress plugins you can install on your site to secure traffic. One of the really handy plugins to do this in WordPress is easy HTTPS SSL redirection. This plugin will let you specify specific pages or posts that need to use HTTPS or you can set it site-wide. On the WordPress admin Dashboard, when you go into the HTTPS redirection under Settings, you can turn on HTTPS site-wide./p>
